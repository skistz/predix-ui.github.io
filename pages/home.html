Predix Design System

Our user interface components enable you to quickly and easily create Industrial Internet web applications that run on dedicated Predix services and data.

Design Starter Kit

A Sketch file that contains core layouts, components, colors and fonts. The Design Starter Kit makes designing a consistent app simple.


Web Components & CSS

The Predix Design System is built using modern web component standards so developers can start using our building blocks with minimal ramp-up.

Framework-agnostic Code

Predix UI components are implemented using modern web application technologies, such as a powerful, scalable and Sass-based CSS framework. You'll find components powered by D3, the best of open source web tech, and we're going all-in on web components. It's framework-agnostic, so you can use Angular, Ember, Backbone, or pure Polymer.

Contribute to the Predix Design System

Do you have a proposed change to and existing component or a brand new component you'd like to contribute back? Learn how to file an issue or make a contribution to the Design System.
